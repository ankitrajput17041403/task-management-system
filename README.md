# Task Management System (SaaS)

A multi-tenant enterprise task and workflow management system built using Spring Boot.

## Features
- Multi-tenant architecture (Organization-based)
- Role-Based Access Control (RBAC)
- JWT Authentication
- Task lifecycle management
- File upload system
- Redis caching
- Docker deployment

## Tech Stack
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- Redis
- Flyway
- Docker
